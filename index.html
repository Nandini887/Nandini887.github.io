Trend Classification and Concept Drift Analysis in TikTok & YouTube Data

December 2025

Project Milestones

  • Problem: Engagement patterns on TikTok and YouTube evolve rapidly, causing concept drift that degrades ML model performance and reduces accuracy in classifying trending vs non-trending videos.
  • Approach: Developed a concept drift detection framework to classify videos as trending or non-trending while monitoring non-stationary engagement data and proactively identifying model performance degradation.
  • Modeling: Trained and benchmarked multiple ML approaches, including streaming/incremental models, adaptive ensemble methods (Adaptive XGBoost, Gradient Boosting), and temporal neural networks.
  • Validation: Applied rolling-window validation and Population Stability Index (PSI) to detect temporal drift and quantify shifts in feature and prediction distributions.
  • Results: Identified consistent performance degradation under non-stationary conditions and found adaptive tree-based ensemble models to be more robust to drift than neural networks.
  • Tools: Python (pandas, scikit-learn, river), NumPy, Matplotlib, Seaborn, GitHub

Impact

This project demonstrates how combining trend classification with concept drift detection helps maintain reliable predictions in rapidly evolving social media environments. The framework provides a practical approach for proactively monitoring and retraining models to prevent silent performance failures in production systems.

MSBA Curriculum vs Industry Skill Demand Analysis

October 2025

Project Milestones

  • Problem Assessment: Investigated whether top global MSBA programs are effectively teaching the skills currently demanded by employers for Data Analyst, Data Scientist, and Business Analyst roles.
  • Data Collection: Scraped MSBA curriculum content from top U.S. and non-U.S. universities and collected live job postings from Indeed to capture current market skill demand.
  • Text Processing & Feature Extraction: Cleaned and preprocessed unstructured curriculum and job description text, extracting skill features using CountVectorizer and NLP-based techniques.

  • Evaluation & Insights: Analyzed alignment gaps, curriculum topic distribution, and ROI comparisons across programs, highlighting mismatches between advertised curricula and real-world job requirements.
  • Tools: Python (pandas, numpy, sklearn, nltk, gensim), CountVectorizer, Cosine Similarity, LDA Topic Modeling, matplotlib, seaborn

Impact

The analysis provides data-driven insights into how well MSBA programs align with evolving industry skill demands, helping prospective students and institutions identify curriculum gaps and optimize program positioning in a competitive analytics job market.
